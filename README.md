# Painter
## Description
A web application for quick drawing, mirrorong https://www.autodraw.com.

## Features
1- Pick colors.
2- Draw: Brush - Shapes - Stickers - Background - Text.
3- Change the canvas size.
4- Redo - Undo - Delete.
5- Share your art - Download your art.
6- How to use tutorial.

## Used Technologies
HTML - CSS - Vanilla JS.
